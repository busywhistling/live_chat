Live chat

Login in the adjacent window to start your live chat session. Choose a room name, and share with the other participant and start chatting. Note that since no cookies are used, your session will be lost if you refresh the page. You can, however, re-join the chat if you remember the room name.

Made by Paramjit (see the rest of the portfolio)
Using express.js, socket.io, TailwindCSS, npm, nodemon & vite. Using vanilla JS for the UI.
